    面试官，你好。我叫三少，我是于 2018 年交通大学本科计算机专业毕业，并且加入了水果公司，主要从事公司电商平台 JAVA 研发的相关工作，至今为止有两年的 JAVA 研发的工作经验，在整个的两年的研发过程当中，我主要负责了电商的微服务项目，承担了其中的商品和交易模块的系统负责人的角色，主要负责开发商品和交易系统。满足整个水果公司电商的商品和交易平台的业务需求。在整个开发的过程当中，我会使用 Spring Boot + Spring Cloud 微服务架构体系去做一个分布式系统的研发，在应对不断变化和迭代的业务需求的过程当中，我采用了抽象的领域模型驱动式的方式，来应对不断变化的需求，保证我们核心系统稳定的能力，作为一个能抗住高并发的系统场景来说，我们已经有了一些结果的产出，我们每年会搞 520，920 的之类的大型促销活动，会有上百万的 DAU 以及上亿级别的 GMV，因此在整个商品和交易的高并发场景内，我们可以在峰值时期抗住 10W 级别的查询 QPS 量，以及可以满足 2000 个并发，2000/QPS，2000/TPS 这样的交易场景的并发，我再整个的项目经历过程当中，主要的技术栈会采用 JAVA 开发语言，一开始我会接触 Dubbo 的 RPC 框架，之后因为公司的技术选型问题，后面全部都切换成了 Spring Boot 和 Spring Cloud 这样更为流行全栈式的微服务框架，并且会使用一些常用的数据中间件，例如：Mysql，Redis 缓存，以及 MongoDB 这种 NoSQL 的数据库，还有最后我们团队使用的是 Docker + K8s 自动化运维工具，我在发布的流程当中也有一些知识的涉列，平时我是一个非常好学的人，除了完成公司的任务之外，我还会额外的去接触一些和自己业务相关或者无关的技术领域，最近我在学习 xxxx
